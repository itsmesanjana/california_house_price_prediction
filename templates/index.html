California Housing Price Prediction System

A machine learning web application deployed using Flask for real-time housing price estimation.

Project Overview

This project demonstrates an end-to-end machine learning workflow, from exploratory data analysis and model training to production deployment through a Flask-based web application.

The system predicts median housing prices in California districts using demographic and geographic features. The trained regression model is integrated into a REST-enabled backend and served through a clean user interface.

Model & Dataset

The model was trained on the California Housing dataset using Linear Regression. Prior to training, complete exploratory data analysis (EDA) was performed, including feature inspection, correlation analysis, and data preprocessing.

Input features include:

  • Median Income
  • House Age
  • Average Rooms
  • Population
  • Average Occupancy
  • Latitude

The objective is to estimate the median house value of a district based on these attributes.

Model Performance

The model was evaluated using standard regression metrics:

  • R² Score: 0.51
  • Mean Squared Error: 0.66
  • Root Mean Squared Error: 0.81

System Architecture

  • Data preprocessing and model training in Python (Scikit-learn)
  • Serialized trained model using joblib
  • Flask backend for inference handling
  • REST API endpoint: /api/predict
  • Frontend built with HTML and CSS

Applications

  • Property value estimation
  • Real estate investment analysis
  • Market trend evaluation
  • Demonstration of ML deployment workflow
